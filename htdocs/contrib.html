Julius 派生版リスト

オリジナルから改良・改造・移植されて公開されている Julius の派生版のリストです．みなさんの手によってオリジナルに無い独自の機能や改善が追加されています．

詳細は以下からどうぞ．

  • Julius for Windows SAPI
  • JuliusLib/JuliusGUI
  • マルチバンド版 Julius

Julius for Windows SAPI --- Microsoft SAPI 用バックエンド

オリジナルのUnix版を元に，Windows用の音声認識バックエンドエンジンとしてオリジナルの拡張・改善を加えた Julius です．Microsoft SAPI-5.1 にほぼ完全に準拠しています．アプリケーションはこのJuliusを Microsoft SAPI 経由で制御できます．またSALTもサポートしています．SAPI/SALT を用いた音声アプリケーションのサンプルも提供されています．

詳細および入手先: Julius for Microsoft SAPI

JuliusLib/JuliusGUI --- ライブラリ版

オリジナルのUnix版をライブラリ化した JuliusLib, およびJuliusをGUI上で実行できるJuliusGUI が公開されています．Windows, Linux, MacOS など様々な環境で動作します．これを用いることで Julius/Julian をアプリケーション内部に組み込むことができます．

詳細および入手先: http://www.itakura.nuee.nagoya-u.ac.jp/people/banno/julius.html

マルチバンド版 Julius

音響特徴ベクトルの各次元から得られる音響尤度にユーザが定義し た重みをつけて音声認識を行うことができるように改造した Julius で す．マルチバンド音声認識や MFT (Missing Feature Theory) に関する 研究にご利用頂けるかと思います．

詳細および入手先: http://www.furui.cs.titech.ac.jp/mband_julius/
