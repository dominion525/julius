動作環境・インストール
  • コンパイル・インストール手順 - Unix
    Julius/Julian をソースからインストールする手順を説明します． Linux や Soraris などの Unix OS 用です．
  • コンパイル手順 - MinGW
    Rev.3.5 から，Windows での開発は，フリーの開発環境の一つである MinGW - Minimalist GNU for Windows へ移行しました．この文書では，MinGW上での Julius のコンパイル方法を説明します．
  • コンパイル手順 - Visual C++
    Julius を Microsoft Visual C++ でコンパイルする方法です．
  • configure オプションの解説
    アルゴリズムのチューニングやマイクのAPI設定は，コンパイル時に configure オプションで指定する必要があります．また，単語グラフ出力など，コンパイル時に指定しないと使えない機能があります．この文書では，Julius/Julian のコンパイル時に使用する configure スクリプトの全オプションについて解説します．

アプリケーション開発者
  • Juliusの使用許諾
    Julius の使用許諾書です．独自の使用条件ですが，基本的に BSD ライセンスに近いフリーライセンスです．詳しくはリンク先をご覧ください．
  • Julian の認識用文法の書き方
    Julius の記述文法版である Julian は，登場しうる入力文のパターンを記述した認識用文法に従って認識を行います．この文書では，Julianで用いる音声認識用文法の作成方法について説明しています．文法の具体的な作成手順やファイルの書式，妥当性のチェック方法などが記されています．またいくつかのサンプル文法も置いてあります．初めて Julian を使う方は，Julian記述文法音声認識キットもあわせてご覧ください．
  • 複数文法の同時認識
    Julian は，複数の文法を同時に用いて認識することができます．この文書では，複数文法の指定方法について説明します．
  • クラスN-gramの使用方法
    Julius オリジナル版でクラスN-gramを使用する方法について説明しています (Rev.3.4以降)．
  • モジュールモード
    Julius/Julian オリジナル版は，音声認識サーバとしてアプリケーションから ネットワーク経由で利用することができます．この文書ではその具体的方法について 解説します．
  • 無効入力の棄却について
    Julius-3.5 は Gaussian Mixture Model (GMM) を用いて環境音識別と棄却を行う 機能を備えており，物音などの雑音や笑い声，咳といった無効な入力を 判別することができます．この文書では機能の詳細と仕様上の注意， GMMの作成方法等について記されています．
  • 単語グラフ形式
    Julius-3.5 以降では，認識結果を単語グラフの形式で出力する機能がサポートされています．この文書では単語グラフ出力を行う方法，出力結果の読み方について解説します．
  • 認識結果の単語信頼度について
    Juliusでは認識結果の単語ごとにその「信頼度」が付与されています（rev.3.4以降）． この文書では，その基本的な仕組みや値の見方について解説します．

音声認識システム開発者

  • Julius の派生バージョン
    Julius は，プラットフォームやアプリケーションへの組み込み方法ごとに， いくつかの派生版が存在します．この文書ではそれらの派生版の違いや ダウンロード方法について記されています．
  • 音声入力について
    Julius はファイル入力・マイク入力・ネットワーク入力など様々なソースからの 音声入力をサポートしています．この文書ではオリジナル版Julius/Julian における 各音声入力の使用方法と音響モデルとの関連，注意点について記されています．
  • MFCCパラメータの設定
    波形入力を正しく認識するためには，サンプリングレートや窓幅， MFCC ベクトルの次元数といった特徴量パラメータを，音響モデルの 学習時パラメータに合わせて設定する必要があります． この文書では，Julius での特徴量パラメータの設定について， HTK と比較しながら説明します．
  • 使用可能なファイル形式とモデル形式
    音響モデル，言語モデル，記述文法，音声ファイルなど，Julius に与える 各種ファイルについて，そのサポートの範囲と制限を詳細に説明しています．
  • トライフォンの扱いと HMMList ファイルについて
    Julius における音素環境依存モデル（トライフォン）の扱い方，および トライフォンモデル使用時に必須である HMMList ファイルについて解説した文書です．
  • Juliusにおける入力の無音区間・休止の扱い
    音声入力においては，多くの場合，発声区間の前後に無音部分が少 量含まれた状態で入力されます．また，発話中の息継ぎなどにより， 文中や単語間に短い休止が含まれます．高い音声認識精度を得るに は，このような発話前後の無音や発話中の休止部分をうまくモデル 化する必要があります． Julius/Julian ではそのような無音部・ 休止部分を扱う方法をいくつか用意しています． この文書ではその方法について説明します．
  • Julius book for Rev.3.3 (PDF, 88ページ)
    Julius の使用方法，各種付属マニュアル，そしてファイルやマイク入力の 仕様に関するいくつかのドキュメントをまとめた文書です．バージョン 3.3 に対応した文書です．

マニュアル・資料
  • ソースドキュメントブラウザ
    Julius のソースコードドキュメント（HTML形式）です． すべての関数および構造体の定義について詳細に説明されています． 型定義や関数呼び出しをリンクでたどれるので，ソースを理解する助けに なるでしょう．

    文書の内容はすべてソースコードにコメントとして埋め込まれています． コメントからドキュメントの生成は， Doxygenを用いました． 英語および日本語で記述されています．

    なお Julian では仕様が若干異なります．Julian はこちらをご覧下さい．

  • オンラインマニュアル
    Julius/Julian および付属ツールのオンラインマニュアル(HTML形式)です． 日本語および英語で提供されています． なおオリジナルのオンラインマニュアルは， nroff 形式で Julius のアーカイブにも含まれています．
