コンパイル・インストール手順 (MinGW)

Julius は rev.3.5 より，Windowsにおけるフリーの開発環境であるMinGW - Minimalist GNU for Windows でのコンパイルに対応しています．MinGW でコンパイルされた実行バイナリは，MinGW がインストールされていない環境でも動作します．

ここでは MinGW および対応シェル環境である MSYS を用いた，Windows用の Julius/Julian のコンパイル手順について解説します．MinGWについてもっと知りたい方は，以下のリンクを参考にしてください．

  • MinGW - Minimalist GNU for Windows
  • MinGWについて頻繁に尋ねられる質問と回答(MinGW FAQ)

MinGWのインストール

必要なものは以下の通りです．どれもなるべく最新のものがよいでしょう．
  • MinGW --- コアツール
  • MSYS --- シェル環境
  • msys-DTK --- 開発支援ツール (perl 等)
下記のリンクも参考にして，インストールしてください．
  • MinGW - Minimalist GNU for Windows
  • C-Compiler Wiki - MinGW/インストール

ライブラリのインストール

文法コンパイラのコアであるmkfaをコンパイルするには flex ライブラリ が必要です．MinGW にはデフォルトでインストールされていないので，別途インストールする必要があります．

また，.gz ファイルを読み込むには zlib ライブラリが必要です．

どちらもソースを入手してコンパイルするのがよいですが，ここで は配布されているパッケージからコピーする方法を示します．以下の手 順で行います．

  1. GnuWin32のFlex のページへ行く．
  2. Download のところにある「Developer files」の Zip アーカイブをダウンロードする．
  3. アーカイブを展開したディレクトリの "lib" フォルダ内にある libfl.aを，MSYS の /mingw/lib ディレクトリにコピーする．
  4. zlib に関しても同様に，Zlibのページの Developer files から libz.a を /mingw/lib, zlib.h を /mingw/include にコピーする．
最終的に，以下のファイルがあれば OK です．
  • /mingw/lib/libfl.a
  • /mingw/lib/libz.a
  • /mingw/include/zlib.h

コンパイル

コンパイルの方法は，Unix の場合と全く同じです．configureスクリプトも動作します．MinGW環境では，マイクAPIとして portaudio，文字セット変換方法として win32 + libjcode が自動的に選択されます．
