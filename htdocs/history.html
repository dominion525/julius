2007/12/19(Wed.)

◆Julius-4.0 リリース

[リリースノート].

Julius-4.0 は，拡張性や柔軟性の確保を目指し，大幅な刷新が行われました． Juliusの内部データ構造の大幅な見直しおよびモジュール化を行った結果，エ ンジン本体のライブラリ化やマルチデコーディングなど多くの先進的機能を実 現しました．主な新機能は以下のとおりです（全てではありません）．
  • エンジン本体のライブラリ化 "JuliusLib"
  • 複数モデルを用いたマルチデコーディングの対応
  • Julius / Julian の一本化
  • 言語モデルや認識処理の動的追加・削除が可能
  • 4-gram 以上の長さのN-gramの対応
  • ユーザ定義言語関数の組込み
  • Confusion network 出力
  • 音声区間検出 (VAD) の強化（GMM・デコーダベース）
  • ツールの機能追加・新ツール追加・メモリ管理改善等

上位互換性を確保しており，Julius-3.x と同様に使うことが出来ます． 認識の精度および速度は，現段階では前バージョン (3.5.3) とほぼ 同じです．以下のドキュメントも合わせて御覧下さい．

  • Rev.4.0 の新機能
  • JuliusLib API リファレンス
  • JuliusLib コールバック一覧
  • Juliusコマンドマニュアル
  • jconf設定ファイルマニュアル

なお，Julius-4の開発版 CVS レポジトリは新たに julius4 と なりましたのでおしらせします（juliusには旧 3.x が入っていま す）．
