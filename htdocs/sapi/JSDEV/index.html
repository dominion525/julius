Julius for SAPI 開発キット

ダウンロード

ver 2.2 開発キット

使用方法

次のものを用意します。
  • Visual Studio .NET
  • Speech SDK 5.1
  • Julius for SAPI 開発キット
  • zlib 1.1.4

Julius for SAPI 開発キットを展開し、JuliusSAPI_Develop 以下に zlib-1.1.4 という ディレクトリで zlib 1.1.4 を展開し、JuliusSAPI.sln を、VisualStudio .NET で開きます。 パッケージに含まれていないプロジェクトもソリューションに登録されていますので、 プロジェクトがない旨のダイアログが表示されますが、無視して OK を押します。

Visual Studio .NET の設定で、Speech SDK 5.1 のインクルードファイルとライブラリファイル を正しく指定します。

構成を ReleaseMinDependency にして、JuliusSAPI をコンパイルします。
